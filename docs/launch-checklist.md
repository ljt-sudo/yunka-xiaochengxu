# 云咖铺子小程序上线清单

## 账号与资质
- 注册并认证微信小程序账号。
- 开通微信支付商户号，并绑定小程序 AppID。
- 配置小程序服务类目、隐私保护指引和用户协议。
- 开通微信云开发环境，记录环境 ID。

## 支付与交易
- 在 `cloudfunctions/api/index.js` 的 `createPayment` 中替换体验版支付参数占位逻辑。
- 在 `paymentCallback` 中接入微信支付通知验签、商户单号校验、金额校验和交易单号保存。
- 用小额真实订单验证：下单、支付、重复回调、取消支付、库存扣减、积分入账、优惠券核销。

## 订阅消息
- 在小程序后台申请优惠券到账、复购提醒、积分变动模板。
- 将模板 ID 写入 `settings.subscribeTemplates`。
- 每次发送前检查用户授权记录，不做非授权群发。

## 商品与运营
- 在云数据库初始化 `products`、`coupons`、`settings`、`admin_users`。
- 如果需要云端购物车，初始化 `carts` 集合；如果只保留本地购物车，订单创建仍会在云函数重新计算金额和库存。
- 商品图片上传云存储后替换 `cover` 和 `detailImages`。
- 配置固定运费、满包邮门槛、客服微信和客服电话。

## 发布验收
- 微信开发者工具预览首页、商品、购物车、结算、订单、会员、优惠券、后台。
- 至少 2 台真机测试弱网、重复点击支付、从分享卡片进入商品详情。
- 提交体验版给内部人员走完整购买链路。
- 微信审核通过后发布正式版。
