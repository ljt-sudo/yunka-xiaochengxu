# 云咖铺子微信小程序 MVP

这是一个微信原生小程序 + 微信云开发的第一版工程骨架，覆盖商城购买、积分抵现、优惠券、订阅消息唤醒、快递发货和轻量商家后台。

## 打开方式
1. 用微信开发者工具打开 `yunkapuzi-miniprogram`。
2. AppID 先使用测试号或真实小程序 AppID。
3. 在开发者工具中开启云开发，并部署 `cloudfunctions/api`。
4. 首次体验可直接使用本地样例数据；真实上线前按 `docs/launch-checklist.md` 完成账号、支付和订阅模板配置。

## 已实现
- 顾客端：首页、商品列表、商品详情、购物车、确认订单、订单详情、订单列表、会员中心、优惠券中心、售后入口。
- 商家后台：商品库存、订单发货、会员积分调整、优惠券创建、订阅消息任务草稿。
- 交易规则：实付 1 元得 1 积分；50 积分抵 1 元；单笔最多抵商品金额 20%；固定运费 + 满包邮。
- 业务接口层：前端统一调用 `common/api.js`，优先走云函数；开发者工具未部署云环境时自动使用本地样例数据预览。
- 云函数能力：登录、商品、购物车、优惠券、订单预计算、订单创建、支付参数入口、支付回调处理、取消订单、发货、积分调整、订阅任务。

## 核心业务边界
- 商品、订单金额、库存、积分、优惠券最终状态以云函数计算和写入为准，前端只提交商品、数量、地址、券和积分使用意图。
- 创建订单只锁定优惠券，不扣库存、不发积分；支付成功回调后才扣库存、核销券、扣/发积分。
- 取消未支付订单会释放已锁定优惠券；已支付订单售后走人工处理。
- 当前 `createPayment` 仍是商户号未配置前的占位入口，接入微信支付后在云函数中替换为真实小程序支付参数生成和回调验签。

## 验证
